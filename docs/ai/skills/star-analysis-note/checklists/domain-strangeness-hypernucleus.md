# Domain checklist — strangeness / hypernucleus

Apply for Λ, K0S, Σ, Ξ, Ω, hypernuclei (³ΛH, ⁴ΛHe, etc.), and related strangeness measurements.

## Topology and reconstruction

- [ ] Weak-decay topology cuts (DCA, decay length, daughter hits) listed with values
- [ ] Impact parameter cuts for primary vs secondary separation
- [ ] Mass window vs full distribution shown (not only narrow signal window)
- [ ] Hypernucleus identification: mass, dE/dx, TOF if used — nuclear ID helper documented

## Background estimation

- [ ] Method stated: mixed event, like-sign, data-driven sidebands, fitting
- [ ] Background shape validated in control regions
- [ ] Feed-down from heavier states considered or justified omission
- [ ] Matter / antimatter backgrounds treated consistently

## Efficiency and acceptance

- [ ] Efficiency from embedding or MC — sample and version recorded
- [ ] Acceptance (phase space, rapidity, pT) separated from efficiency where possible
- [ ] Closure: MC reconstructed yield vs truth
- [ ] Centrality- and pT-dependent efficiency maps applied or variation quoted

## Physics consistency

- [ ] Known mass positions (Λ, K0S, …) consistent with calibration
- [ ] Yield trends vs centrality / pT physically plausible
- [ ] Coalescence or thermal model comparison if claimed (not over-interpreted)
- [ ] Statistical significance: signal/background ratio and significance method stated

## Systematics (strangeness-specific)

- [ ] Topological cut variation (DCA, cosθ, decay length)
- [ ] Mass window / fit range
- [ ] PID on daughters (proton, pion)
- [ ] Efficiency scale uncertainty
- [ ] Feed-down systematic
- [ ] Event-plane or flow correction (if v_n measured)

## Cross-checks (minimum)

- [ ] Mixed-event or like-sign background comparison
- [ ] Mass window scan
- [ ] Embedding closure
- [ ] Split by run period or half-field if statistics allow
- [ ] Compare to STAR / world data at same energy (qualitative minimum)

## Probing questions

- Is the background subtracted where the signal is zero in MC?
- Are hypernucleus counts limited by PID or by topology?
- Does anti-particle spectrum match particle within systematics?
- Is significance computed after full systematic budget?
